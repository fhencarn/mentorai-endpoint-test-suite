# Labeling System

## Labels

- `working` — endpoint works as expected
- `working-needs-docs` — endpoint works, but docs are unclear or incomplete
- `permission-restricted` — valid endpoint, but restricted
- `syracuse-permission-issue` — appears limited by Syracuse-specific tenant or role permissions
- `bad-request` — payload, parameter, or request formatting issue
- `undocumented` — endpoint appears to exist but is not clearly documented
- `deprecated` — route appears old or replaced
- `server-error` — backend returned a 5xx response
- `unknown` — result needs manual review
